# askdeen

Get started by running the following in the root directory:

```bash
pnpm sst dev
```

## Packages

### Webapp

`/packages/site` - Standard NextJS 14 site

Run with:

```bash
pnpm run dev
```

### API Gateway

`/packages/functions` - AWS API Gateway with Lambdas

Runs with above command `pnpm sst dev`
